창	뜻
Project	프로젝트 전체 파일·에셋
Hierarchy	현재 씬 안의 GameObject 목록
Scene	씬을 편집하고 둘러보는 3D 공간
Inspector	선택한 오브젝트의 상세 설정
Game	실제 플레이어가 보는 테스트 화면
기본 작업 흐름

Project에서 씬 열기 → Hierarchy에서 오브젝트 찾기 → Scene에서 확인·수정 → Inspector에서 값 조절 → Play로 테스트 → Stop → Ctrl+S 저장

꼭 외울 조작
기능	조작
화면 둘러보기	우클릭 + 드래그
자유 이동	우클릭 누른 채 WASD
위 / 아래 이동	우클릭 누른 채 E / Q
오브젝트 화면 중앙 맞추기	F 또는 Hierarchy 더블클릭
확대·축소	마우스 휠
오브젝트 주위 회전	Alt + 좌클릭 드래그
게임 테스트 시작	▶ Play
게임 테스트 종료	■ Stop
저장	Ctrl + S
되돌리기	Ctrl + Z
Inspector에서 제일 중요한 것
Transform
Position = 위치
Rotation = 회전
Scale = 크기
Rigidbody = 물리
Collider = 충돌
Script = 코드로 만든 동작
헷갈리기 쉬운 것
Scene 뷰: 만드는 사람이 보는 편집 화면
Game 뷰: 플레이어가 보는 실제 게임 화면
Play 모드에서 바꾼 것: Stop 누르면 사라짐
씬 이름 옆 *: 아직 저장 안 함
3D 이동 핵심
우클릭 드래그 = 내 자리는 그대로, 고개만 돌림
Alt + 좌클릭 드래그 = 오브젝트 주위를 빙글 돎
Perspective = 가까운 건 크고 먼 건 작음
Isometric = 원근감 없이 위치 확인 쉬움
